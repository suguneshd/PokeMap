# PokéMap
PokéMap 

Project Overview:
PokéMap Explorer is a web application that allows users to discover and explore Pokémon from different generations. Users can search for Pokémon, filter results based on type, and sort them according to different stats. The application uses data from a public API to display detailed Pokémon information including stats, abilities, and images.

Purpose:
The goal of this project is to demonstrate API integration, data handling, filtering and sorting logic, and building a clean and interactive user interface.

Features:
Core Features:
	•	Search Pokémon by name
	•	Display Pokémon name, type, stats, and image

Filtering:
	•	Filter by Pokémon type (Fire, Water, Grass, etc.)

Sorting:
	•	Sort by HP (ascending/descending)
	•	Sort by Attack (ascending/descending)
	•	Sort by Speed (ascending/descending)

API Used:
PokeAPI is used to fetch Pokémon data including name, stats, abilities, types, and images.

Tech Stack:
Frontend: HTML, CSS, JavaScript
